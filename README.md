# AI Agent Project

A simple AI agent built as part of the [boot.dev](https://www.boot.dev) back-end developer curriculum.

> ⚠️ **Disclaimer**: This is not a production-ready agent. It is a small learning project built to explore and understand the fundamentals of building an AI agent from scratch.

## What it can do
- List directory contents
- Read file content
- Write file content
- Run Python files

## Getting Started

### Prerequisites
- Python 3.x
- [uv](https://github.com/astral-sh/uv)

### Installation
```bash
git clone https://github.com/andymucyo-ops/Ai-agent-project.git
cd Ai-agent-project
uv sync
```

### Run
```bash
uv run main.py "<instructions>"
```
